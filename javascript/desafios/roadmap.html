🐍 Roadmap Python para Iniciantes

Aprenda do zero com materiais 100% gratuitos – guia completo e prático

🧰 Fase 0 – Preparação

  • ✅ Instalar Python em python.org
  • ✅ Escolher um editor:
    VS Code (recomendado) • Thonny (para iniciantes) • Google Colab (navegador)
  • ✅ Criar uma pasta de projetos no computador

🚀 Fase 1 – Fundamentos Básicos (2 a 4 semanas)

📚 Tópicos

  • Hello World, variáveis, tipos (int, float, string, bool)
  • input() / print()
  • Operadores aritméticos, lógicos, comparação
  • Condicionais: if, elif, else
  • Laços: for e while

🎓 Materiais gratuitos

  • Curso em Vídeo – Python (Gustavo Guanabara)
    🔗 Playlist completa (GRÁTIS)
  • Python para Zumbis – pynaut.com
  • W3Schools Python – w3schools.com/python

🧠 Fase 2 – Estruturas de Dados e Funções (3 a 4 semanas)

  • Listas, tuplas, dicionários, conjuntos
  • Métodos: append(), remove(), keys(), values()
  • Funções (def, parâmetros, return, escopo)
  • List comprehensions
  • Tratamento de erros (try/except)

🎓 Materiais gratuitos

  • Python.org Tutorial Oficial – docs.python.org
  • Exercism – Python Track – exercism.org
  • CodingBat Python – codingbat.com

🛠️ Fase 3 – Arquivos e Módulos (1 a 2 semanas)

  • Ler/escrever arquivos .txt e .csv
  • Módulos nativos: math, random, datetime, os, json
  • Importação de módulos
  • Real Python – File Handling – realpython.com

🚧 Fase 4 – Projetos Práticos (faça enquanto aprende)

  • 🔢 Calculadora simples
  • 🔐 Gerador de senhas
  • 🎮 Jogo da forca
  • ✅ Lista de tarefas (terminal)
  • 💱 Conversor de moedas com API

📌 Ideias e desafios

  • FreeCodeCamp – 100+ projetos Python
  • Python Practice Book (GitHub)
